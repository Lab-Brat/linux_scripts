## Set Up
Clone the repository and navigate to it
```bash
git clone https://github.com/Lab-Brat/linux_scripts.git
cd linux_scripts
```  

Then, install virtual enironment and build the app with setup.py
```bash
virtualenv venv
. venv/bin/activate
pip install --editable .
admin
```
